# we
A friendly TypeScript CLI :)

### Development
* Pull requests are always welcome. Pick an issue and help us out to make this CLI awesome!
  
###	Development guide
* Clone the project:
  * ```$ git clone https://github.com/USERNAME/we.git```
* Go the project:
  * ```$ cd we```
* Install dependencies:
  * ```$ npm install```
* Start and have fun!
  * ```$ npm start```
